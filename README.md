# Uruguay XXI × Eduba landing page

Intended URL path: /for/uruguay-xxi

Prospect-facing page for Uruguay XXI, the Uruguayan investment, export and country-brand promotion agency, prepared for eMerge Americas 2026 (booth 352). The page is Spanish-primary with an ES/EN toggle in the institutional top strip, matching the trilingual convention of uruguayxxi.gub.uy. It carries the fit-analysis pitch body: a "read" on where AI fits in the 45-company delegation and 500-exporter catalog coordination work, the 60/30/10 frame applied to Uruguay XXI's own workflows, the Correlation One case study, the Interpretable Context Methodology paper, and a direct booking link to Matt Creamer's calendar.

Visual language echoes the agency's own institutional IPA aesthetic: navy primary (#00549C) with the Uruguayan sun motif (navy and yellow concentric circles) as a subtle hero element, sky-blue secondary (#1db7eb), sun yellow (#FFC341) as a rare accent, Inter throughout, generous whitespace. No Eduba branding at the top. Eduba surfaces only in the footer line and in the named case study.
